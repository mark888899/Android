มีอะไรใหม่ในแอพเมล

แอพเมลใน macOS Sierra มีคุณสมบัติใหม่ๆ ที่โดดเด่นในการส่ง รับ และดูอีเมล

เปิดเมลให้ฉัน

Siri

Siri ใน Mac ของคุณ ช่วยให้การใช้แอพเมลเป็นเรื่องง่ายขึ้นกว่าเดิมคุณสามารถส่งอีเมลได้อย่างรวดเร็ว ขอให้ Siri อ่านอีเมลให้คุณฟัง สร้างการเตือนความจำเพื่อดูอีเมลในภายหลัง ค้นหาอีเมล และอื่นๆตัวอย่างต่อไปนี้เป็นสิ่งที่คุณสามารถขอให้ Siri ช่วยคุณได้:

  • “ส่งอีเมลถึง John เกี่ยวกับแผนการท่องเที่ยวของเรา”

  • “อ่านอีเมลให้ฉันฟัง”

  • “เตือนฉันเกี่ยวกับเรื่องนี้ในอีกหนึ่งชั่วโมง”

  • “ค้นหาอีเมลเกี่ยวกับการปรับปรุงห้องครัว”

ฟิลเตอร์

ค้นหาอีเมลในกล่องเมลโดยใช้ฟิลเตอร์เพื่อเลือกข้อความที่ต้องการให้แสดงตัวอย่างเช่น ปรับใช้ฟิลเตอร์เพื่อให้แสดงเฉพาะข้อความที่ยังไม่ได้อ่านหรือข้อความที่มีไฟล์แนบ หรือข้อความที่เป็นไปตามเกณฑ์ทั้งสองคุณสามารปรับใช้ฟิลเตอร์ได้สูงสุดหกฟิลเตอร์

การสนทนา

การสนทนามีการจัดให้เป็นระเบียบ โดยบรรทัดชื่อเรื่องจะแสดงอยู่ด้านบนสุดของการสนทนาเท่านั้น แทนที่จะเป็นข้อความแต่ละข้อความ เพื่อให้อ่านได้ง่ายขึ้น

หากคุณยังไม่คุ้นกับแอพเมล ข้อความล่าสุดในการสนทนาจะปรากฏอยู่ด้านล่างสุด คุณจึงสามารถติดตามประเด็นจากข้อความแรกลงไปยังข้อความล่าสุดได้ในกรณีอื่นๆ การเรียงลำดับของข้อความในการสนทนาจะไม่เปลี่ยนแปลงไปจากแอพเมลใน OS X v10.11

การตั้งค่าบัญชี

การตั้งค่าบัญชีในแอพเมลมีการจัดระเบียบใหม่ คุณจึงสามารถหาตัวเลือกที่คุณอาจเปลี่ยนบ่อยได้ง่ายขึ้น เช่น นามแฝงอีเมล หรือวิธีการจัดการกับไฟล์แนบตัวเลือกสำหรับกล่องเมลขยะ ถังขยะ และกล่องเมลอื่นๆ มีการรวมเข้าไว้ด้วยกัน

คุณสมบัติอื่นๆ

  • สำหรับบัญชี Exchange แอพเมลจะตรวจหาข้อความใหม่โดยอัตโนมัติหากเซิร์ฟเวอร์เมลใช้การแจ้งเตือนแบบสตรีมของ Exchange Web Services (EWS)

  • หากคุณใช้บัญชีอีเมลหลายบัญชี แอพเมลจะช่วยคุณเลือกที่อยู่อีเมลที่ต้องการในช่องจากเมื่อคุณส่งข้อความ โดยจะเลือกตามที่อยู่อีเมลของผู้รับรายแรกในช่องถึงตัวอย่างเช่น หากที่อยู่อีเมลหลัก (ในการตั้งค่าการเขียน) เป็นที่อยู่ Gmail ส่วนบุคคลของคุณ แต่คุณกำลังเขียนอีเมลถึงอาจารย์ซึ่งใช้ที่อยู่อีเมลของโรงเรียน (.edu) แอพเมลจะใช้ที่อยู่อีเมลของโรงเรียนแทนที่จะใช้ที่อยู่อีเมลส่วนบุคคลของคุณในช่องจาก

  • หากคุณตั้งค่าตัวเลือก (ในการตั้งค่าการเรียกดู) ให้ย้ายข้อความที่ละทิ้งไปยังกล่องเมลจัดเก็บ คุณสามารถจัดเก็บข้อความถาวรได้โดยตรงจากการแจ้งเตือนของเมล

  • คุณสามารถใช้แถบในแอพเมลได้ แม้ว่าคุณจะไม่ได้ใช้แอพเมลแบบเต็มหน้าจอ

  • แอพเมลช่วยเพิ่มพื้นที่ว่างใน Mac ของคุณด้วยตัวเลือก (ในการตั้งค่าบัญชี) ที่ให้คุณระบุไฟล์แนบที่คุณต้องการให้ดาวน์โหลดโดยอัตโนมัติเมื่อคุณอ่านอีเมล โดยแบ่งเป็นไฟแนบทั้งหมด ล่าสุด หรือไม่เลือกไฟล์แนบใดๆ
