Novidades no Mail

A aplicação Mail no macOS Sierra tem muitas e importantes funcionalidades novas para enviar, receber e ver e-mails.

Abrir o Mail

Siri

Siri no Mac faz com que seja ainda mais fácil usar o Mail. Pode enviar um e-mail rápido, fazer com que o e-mail seja lido para si, criar um lembrete para ver um e-mail mais tarde, pesquisar e-mails, etc. Eis alguns exemplos de coisas que pode dizer a Siri:

Filtros

Encontrar rapidamente e-mail numa caixa de correio utilizando filtros para alterar que mensagens são mostradas. Por exemplo, aplique um filtro para mostrar apenas mensagens não lidas ou que têm anexos — ou mensagens que cumprem ambos os critérios. Pode aplicar até seis filtros.

Conversas

As conversas são simplificadas — o assunto é mostrado apenas no topo da conversa, em vez de aparecer em cada mensagem — para facilitar a leitura.

Se o Mail é uma novidade para si, a mensagem mais recente numa conversa é mostrada no fundo, para que possa acompanhar a conversa a partir da primeira mensagem em baixo até à mais recente. Caso contrário, a ordem das mensagem em conversas mantêm-se inalterada desde que usava o Mail no OS X v10.11.

Preferências de contas

O painel de preferências Contas no Mail foi reorganizado, para que seja mais fácil encontrar opções que pode querer alterar mais frequentemente como, por exemplo, alias de e-mail ou como gere os anexos. As opções para correio indesejado, lixo e outras caixas do correio foram consolidadas.

Outras funcionalidades

  • Para contas Exchange, o Mail procura automaticamente novas mensagens se o servidor de correio da conta usar notificações Exchange Web Services (EWS).

  • Se usar várias contas de e-mail, o Mail pode garantir que usa o endereço De pretendido ao enviar uma mensagem, com base no endereço de e-mail do primeiro destinatário no campo Para. Por exemplo, se o seu endereço de e-mail preferido (no painel de preferências Composição) for a sua conta Gmail pessoal, mas estiver a escrever um e-mail para o seu professor para o endereço escolar (.edu), o Mail usa o seu endereço escolar em vez do seu endereço pessoal no campo De.

  • Se definir a opção (no painel de preferências Visualização) para mover as mensagens descartadas para a caixa de correio Arquivo, pode arquivar mensagens diretamente a partir das notificações do Mail.

  • Pode utilizar separadores nas janelas do Mail, mesmo quando não está a usar o Mail em ecrã completo.

  • O Mail ajuda-o a otimizar o espaço de armazenamento no Mac fornecendo uma opção (no painel de preferências Contas) para especificar que anexos — todos, recentes ou nenhum — são descarregados automaticamente ao ler o e-mail.
