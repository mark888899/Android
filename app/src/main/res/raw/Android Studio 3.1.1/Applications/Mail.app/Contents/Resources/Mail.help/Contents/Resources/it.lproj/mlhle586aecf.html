server di posta

Un computer gestito da un provider di account e-mail che funge da ufficio postale digitale e consente il trasferimento di messaggi e-mail dal mittente al destinatario. I server di posta in uscita (spesso chiamati server SMTP) gestiscono i messaggi inviati. I server di posta in entrata gestiscono i messaggi ricevuti. Per gli account IMAP (come iCloud o Gmail), le copie dei messaggi vengono archiviate sui server di posta in entrata fino a quando non vengono eliminati.
