「郵件」的新功能

macOS Sierra 中的「郵件」App 提供了收發和檢視郵件時值得注意的新功能。

為我打開「郵件」

Siri

Mac 上的 Siri 可讓使用「郵件」更為簡單。您可以傳送快速郵件、讓郵件朗讀給您聽、製作提醒事項來稍後閱讀郵件、搜尋郵件等等。以下是您可以對 Siri 說的例子：

  • 「寄信給哥哥討論旅行計畫」

  • 「朗讀我的郵件」

  • 「1 小時後提醒我」

  • 「尋找廚房改裝的電子郵件」

過濾條件

使用過濾條件來更改要顯示的郵件，以快速在信箱中尋找郵件。例如，套用過濾條件來僅顯示未讀郵件或夾帶附件的郵件，或是同時符合這兩個條件的郵件。您最多可以套用六個過濾條件。

對話

對話具流暢性：標題行只會顯示在對話最上方，而不會顯示在每風郵件內，因此更易於閱讀。

若您為「郵件」使用新手，最話中最近的郵件會顯示在底部，因此您可以沿著討論串從第一封郵件依序往下看到最新的郵件。否則，對話中郵件的順序會維持 OS X v10.11 中「郵件」的排序方式。

「帳號」偏好設定

「郵件」中的「帳號」偏好設定已重整過，讓您可以更容易找到您可能會經常更改的選項，例如電子郵件替身或處理附件的方式。已統整垃圾郵件、垃圾桶和其他信箱的選項。

其他功能

  • 針對 Exchange 帳號，如果帳號的郵件伺服器是使用 Exchange Web Services（EWS）串流通知，則「郵件」會自動檢查新郵件。

  • 若您使用多個電子郵件帳號，「郵件」可協助您在傳送郵件時，依據「收件人」欄位中的第一個收件人電子郵件位址，確保您使用的是正確的「寄件人」位址。例如，若您偏好的電子郵件位址（在「編寫郵件」偏好設定中）是您個人的 Gmail 位址，但您打算寫一封郵件給教授的學校信箱（.edu），「郵件」會在「寄件人」欄位中使用您的學校位址，而不會使用您的個人位址。

  • 若您設定選項（在「檢視」偏好設定中）來將不要的郵件移到「封存」信箱，您可以直接從「郵件」通知封存郵件。

  • 您可以在「郵件」視窗中使用標籤頁，甚至在您未以全螢幕使用「郵件」時。

  • 「郵件」可協助您最佳化 Mac 上的儲存空間，其提供了一個選項（在「帳號」偏好設定中）來指定當您閱讀郵件時要自動下載哪些附件（全部、最近或無）。
