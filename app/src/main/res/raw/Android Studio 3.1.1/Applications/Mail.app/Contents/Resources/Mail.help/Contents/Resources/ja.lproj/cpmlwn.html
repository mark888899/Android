メールの新機能

macOS Sierra の「メール」アプリケーションには、メールの送受信および表示に関する注目の新機能が追加されています。

「メール」を開く

Siri

Mac の Siri によって、「メール」をより簡単に使用できるようになりました。簡単なメールを送信したり、受信したメールを読み上げたり、メールを後で確認するようにリマインダーを作成したり、メールを検索したり、さまざまなことを実行できます。Siri への話しかけかたの例を以下に示します：

  • 「一郎に旅行のプランに関するメールを送信して」

  • 「メールを読み上げて」

  • 「これについて 1 時間後にリマインドして」

  • 「キッチンのリモデルに関するメールを検索して」

フィルタ

フィルタを適用して表示するメッセージを絞り込むことで、メールボックス内のメールをすばやく見つけることができます。たとえば、未読メッセージのみ、添付ファイル付きのメッセージのみ、またはこの両方に当てはまるメッセージのみを表示するフィルタを適用できます。フィルタは最大 6 個まで適用できます。

スレッド

スレッドがさらに使いやすくなりました。メッセージごとではなくスレッドの上部のみに件名が表示されるので、より一層見やすくなっています。

「メール」をはじめて使用する場合は、スレッド内の一番新しいメッセージは一番下に表示されるので、スレッドの最初のメッセージから下に移動すると最新のメッセージまでたどることができます。そうでない場合は、スレッド内のメッセージの順序は OS X v10.11 の「メール」の設定のままになります。

「アカウント」環境設定

「メール」の「アカウント」環境設定の構成が改善され、メールのエイリアスや添付ファイルの処理方法など、頻繁に変更するオプションを見つけやすくなりました。迷惑メールやゴミ箱などのメールボックスのオプションが統合されました。

その他の機能

  • Exchange アカウントのメールサーバが Exchange Web Services（EWS）ストリーミング通知を使用している場合は、新着メッセージが自動的にチェックされます。

  • 複数のメールアカウントを使用している場合は、メッセージを送信するときに、「宛先」フィールドの最初の受信者のメールアドレスに基づいて目的の「差出人」アドレスが自動的に使用されます。たとえば、「作成」環境設定の優先メールアドレスが個人の Gmail アドレスになっているときに、教授の大学のアドレス（.edu）宛にメールを書いている場合は、個人のアドレスではなく大学のアドレスが「差出人」フィールドで自動的に使用されます。

  • 「表示」環境設定のオプションで、削除したメッセージを「アーカイブ」メールボックスに移動するように設定した場合は、「メール」の通知から直接メッセージをアーカイブできます。

  • 「メール」をフルスクリーンで使用していないときでも「メール」ウインドウのタブを使用できます。

  • 「アカウント」環境設定のオプションで、メールを読むときに自動的にダウンロードする添付ファイル（すべて、最近、なし）を指定することで、Mac 上のストレージを最適化できます。
