Mail의 새로운 기능

macOS Sierra의 Mail 앱은 이메일을 주고 받고, 보는 데 유용한 주목할 만한 새로운 기능을 제공합니다.

Mail 열기

Siri

Mac에서 Siri를 사용하면 Mail 사용이 훨씬 쉬워집니다. 빠르게 이메일을 보낼 수 있고, 이메일을 읽어주도록 할 수 있으며, 나중에 이메일을 볼 수 있도록 미리 알림을 생성할 수도 있고, 이메일을 검색하는 등 여러 작업을 수행할 수 있습니다. 다음은 Siri에서 말할 수 있는 몇 가지 예제입니다.

  • “Send an email to John about our travel plans”

  • “Read my email”

  • “Remind me about this in an hour”

  • “Find emails about kitchen remodel”

필터

필터를 사용하여 메시지가 표시되는 방식을 변경하면 메일상자에서 이메일을 빨리 찾을 수 있습니다. 예를 들어, 읽지 않은 메시지나 첨부 파일이 있는 메시지가 표시하거나 두 기준을 모두 충족하는 메시지만 표시하도록 필터를 적용하십시오. 최대 6개의 필터를 적용할 수 있습니다.

대화

제목이 각 메시지가 아닌 대화 상단에만 표시되어 대화가 간결해지고 읽기 쉬워졌습니다.

Mail을 처음 사용하는 경우, 대화의 최근 메시지가 맨 아래에 표시됩니다. 스레드를 따라 내려가며 처음 받은 메시지부터 마지막 메시지까지 확인할 수 있습니다. OS X v10.11에서 Mail을 사용한 경우에는 대화의 메시지 순서가 사용한 그대로 표시됩니다.

계정 환경설정

Mail의 계정 환경설정이 재구성되어 이메일 구성본 또는 첨부 파일 처리 방법 등과 같이 자주 변경하는 옵션을 더 쉽게 찾을 수 있습니다. 정크 메일, 휴지통 및 기타 메일상자의 옵션이 통합되었습니다.

다른 기능

  • 계정 메일 서버가 EWS(Exchange 웹 서비스 계정) 스트리밍 알림을 사용하는 경우, Mail은 자동으로 Exchange 계정의 새 메시지를 확인합니다.

  • 사용자가 여러 이메일 계정을 사용하는 경우, Mail은 사용자가 메시지를 보낼 때 받는 사람 필드에 있는 첫번째 수신자의 이메일 주소를 기반으로 적절한 보내는 사람 주소를 사용하도록 표시합니다. 예를 들어, 사용자의 선호하는 이메일 주소(작성 환경설정에서)가 개인용 Gmail 주소이지만 학교 주소(.edu)를 사용하는 교수에게 이메일을 작성하는 경우, Mail은 보내는 사람 필드에 사용자의 개인용 주소가 아닌 학교용 주소를 표시합니다.

  • 무시한 메시지를 아카이브 메일상자로 이동하도록 옵션(보기 환경설정)을 설정하면 Mail 알림에서 바로 메시지를 아카이브할 수 있습니다.

  • Mail 전체 화면을 사용하고 있지 않아도 Mail 윈도우의 탭을 사용할 수 있습니다.

  • Mail은 사용자가 이메일을 읽을 때 자동으로 다운로드할 첨부 파일이 무엇인지 모두, 최근, 없음 중에서 지정할 수 있는 옵션(계정 환경설정)을 제공하여 사용자의 Mac 저장 공간을 최적화합니다.
