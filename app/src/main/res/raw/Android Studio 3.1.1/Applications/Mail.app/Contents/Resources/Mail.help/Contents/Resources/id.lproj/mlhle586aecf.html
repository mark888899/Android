server mail

Komputer yang dikelola dengan penyedia akun email yang bertindak sebagai kantor pos digital dan mengizinkan transfer pesan email dari pengirim ke penerima. Server mail keluar (sering disebut server SMTP) menangani pesan yang sedang dikirim. Server mail masuk menangani pesan yang sedang diterima. Untuk akun IMAP (seperti iCloud atau Gmail), salinan pesan disimpan di server mail masuk hingga pesan dihapus.
