Nouveautés de Mail

Dans macOS Sierra, Mail propose de nouvelles fonctionnalités remarquables pour envoyer, recevoir et afficher vos e-mails.

Ouvrir Mail

Siri

Siri sur votre Mac simplifie l’utilisation de Mail. Vous pouvez envoyer un e-mail rapidement, vous faire lire vos e-mails, créer un rappel afin de lire un e-mail plus tard, rechercher un e-mail, et plus encore. Voici quelques exemples de ce que vous pouvez dire à Siri :

  • “Envoie un e-mail à Jean au sujet de notre projet de voyage”

  • “Lis-moi mes e-mails“

  • “Rappelle-moi ça dans une heure“

  • “Trouve les e-mails concernant la rénovation de la cuisine“

Filtres

Retrouvez facilement des e-mails dans une boîte aux lettres à l’aide de filtres pour modifier les messages qui sont affichés. Par exemple, appliquez un filtre pour afficher uniquement les messages non lus ou ceux avec des pièces jointes, ou les messages remplissant ces deux critères. Vous pouvez appliquer jusqu’à six filtres.

Conversations

Les conversations sont simplifiées : la ligne d’objet est uniquement affichée en haut de la conversation plutôt que dans chaque message, ce qui en facilite la lecture.

Si vous découvrez Mail, le message le plus récent d’une conversation s’affiche en bas, afin de pouvoir suivre le fil du premier message jusqu’au plus récent. Autrement, l’ordre des messages dans une conversation reste inchangé depuis Mail dans OS X 10.11.

Préférences Comptes

Les préférences Compte dans Mail ont été réorganisées, il est maintenant plus simple de trouver les options que vous changerez fréquemment, comme les alias d’e-mail ou la gestion des pièces jointes. Les options pour le courrier indésirable, la corbeille et les autres boîtes aux lettres ont été consolidées.

Autres fonctionnalités

  • Pour les comptes Exchange, Mail recherche automatiquement les nouveaux messages si le serveur de messagerie du compte utilise les notifications streaming Services Web Exchange (EWS).

  • Si vous utilisez plusieurs comptes de messagerie, Mail vous permet de vous assurer que vous utilisez l’adresse d’expédition souhaitée quand vous envoyez un message, en se basant sur l’adresse e-mail de votre premier destinataire dans le champ À. Par exemple, si votre adresse e-mail préférée (dans les préférences Rédaction) est votre adresse personnelle Gmail, mais que vous écrivez un e-mail à votre professeur à son adresse universitaire (.edu), Mail utilise votre adresse universitaire plutôt que votre adresse personnelle dans le champ De.

  • Si vous définissez l’option (dans les préférences Présentation) consistant à déplacer les messages supprimés dans la boîte Archives, vous pourrez archiver les messages directement depuis les notifications Mail.

  • Vous pouvez utiliser les onglets dans les fenêtres de Mail, même lorsque vous n’utilisez pas Mail en mode plein écran.

  • Mail vous aide à optimiser l’espace de stockage sur votre Mac en proposant une option (dans les préférences Comptes) qui permet de spécifier quelles pièces jointes (toutes, récentes ou aucune) doivent être téléchargées automatiquement quand vous consultez vos e-mails.
