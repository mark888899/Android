연결 해결사 사용하기

Mail 연결 해결사를 사용하여 인터넷 연결 또는 이메일 계정 상태를 점검하십시오.

참조: 계정 또는 네트워크가 오프라인이면 번개 모양 아이콘 또는 경고 아이콘이 검색 필드 바로 아래 있는 즐겨찾기 막대 끝이나 Mail 사이드바에 나타납니다. 가능한 경우 번개 모양 아이콘을 클릭하여 계정을 온라인으로 만들거나 상태 정보를 확인합니다.

Mail 열기

  1. 윈도우 > 연결 해결사를 선택하십시오.

  2. 상태 정보를 확인한 다음 문제를 해결하십시오.

    • 인터넷 연결 상태 확인하기: 윈도우 상단의 연결 상태가 빨간색 점으로 보이는 경우 Mail이 인터넷에 연결되지 않은 것입니다. 네트워크 진단을 클릭하여 문제를 해결하는 데 도움을 받으십시오.

    • 이메일 계정 확인하기: 계정이 빨간색 상태 점으로 표시되는 경우(인터넷에 연결되어 있을 때) 계정 제공업체의 메일 서버 또는 계정 설정에 문제가 있을 수 있습니다. 세부사항 열에 보이는 지시사항에 따르십시오. Mail 환경설정의 계정 패널에서 계정을 보려면 세부사항 열에 있는 텍스트를 이중 클릭하십시오.

    Mail을 연결하려고 시도하는 동안 로그 정보를 보려면 세부사항 보기를 클릭한 다음 다시 확인을 클릭하십시오.

  3. 완료했다면, 다시 확인을 클릭하여 사용자의 동작이 문제를 해결했는지를 확인하십시오.

    문제가 지속되면 계정 제공업체에 문의하여 계정 세부사항을 확인하고 설정하십시오.

Mail 문제와 관련하여 Apple 지원에 연락하는 경우 윈도우의 로그 연결 진행 상태를 선택하도록 요청받을 수 있습니다. 로그 연결 진행 상태를 선택하면 Apple 지원이 사용자 컴퓨터의 Mail 문제를 파악 및 조사하고 Apple 제품과 서비스를 개선할 수 있도록 사용자가 Mail을 사용할 때 진단 정보를 수집합니다. 수집된 진단 정보는 사용자의 이메일 메시지와 같은 개인 정보를 포함할 수 있습니다. 개인정보 취급방침에 따라 Apple은 전송받은 진단 정보를 다른 회사와 공유하지 않습니다.

Finder에서 로그 파일이 있는 폴더를 보려면 연결 해결사 윈도우에서 로그 보기를 클릭하십시오. Apple 지원에 로그 파일을 제공한 다음 체크상자를 선택 해제하여 연결 진행 상태 로그 기록을 끌 수 있습니다. 로그 파일의 크기가 커지면 삭제하고 싶을 수 있습니다.

www.apple.com/kr/privacy에서 Apple 개인정보 취급방침에 대해 더 알아볼 수 있습니다.
