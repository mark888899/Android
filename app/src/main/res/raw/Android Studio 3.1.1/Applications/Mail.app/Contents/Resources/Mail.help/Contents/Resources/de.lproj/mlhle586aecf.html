Mail-Server

Ein von einem E-Mail-Accountanbieter verwalteter Computer, der als digitales Postamt fungiert und dafür sorgt, dass E-Mails vom Absender an den Empfänger übertragen werden. Server für ausgehende E-Mails (oft SMTP-Server genannt) verwalten die Nachrichten, die versendet werden. Server für eingehende E-Mails verwalten Nachrichten, die empfangen werden. Bei IMAP-Accounts (wie iCloud oder Gmail) werden Kopien der E-Mails auf den Servern für eingehende E-Mails gespeichert, bis sie gelöscht werden.
