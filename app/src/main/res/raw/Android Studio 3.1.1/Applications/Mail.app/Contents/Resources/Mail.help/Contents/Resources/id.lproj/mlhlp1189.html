Menggunakan Dokter Koneksi

Gunakan Dokter Koneksi Mail untuk memeriksa koneksi Internet Anda atau status akun email Anda.

Kiat: Saat jaringan akun Anda offline, ikon petir atau ikon peringatan akan muncul di ujung bar Favorit (di bawah bidang pencarian) atau di bar samping Mail. Anda dapat mengeklik ikon untuk menjadikan akun online, jika memungkinkan, atau untuk melihat informasi status.

Buka Mail untuk saya

  1. Pilih Jendela > Dokter Koneksi.

  2. Tinjau informasi status, lalu coba atasi masalah.

    • Periksa koneksi Internet Anda: Jika Status Koneksi di bagian atas jendela menunjukkan titik merah, Mail tidak dapat terhubung ke Internet. Klik Diagnostik Jaringan untuk memperoleh bantuan penyelesaian masalah.

    • Periksa akun email Anda.: Jika akun menunjukkan titik status berwarna merah (saat Anda terhubung ke Internet) mungkin ada masalah pada server mail penyedia akun, atau pengaturan akun Anda. Ikuti petunjuk yang ditampilkan pada kolom Detail. Untuk melihat akun di panel Akun pada preferensi Mail, klik teks dua kali pada kolom Detail.

    Untuk melihat informasi log saat Mail mencoba untuk terhubung, klik Tampilkan detail, lalu klik Periksa Kembali.

  3. Setelah selesai, klik Periksa Lagi untuk melihat apakah tindakan Anda mengatasi masalah.

    Jika masalah masih terjadi, hubungi penyedia akun Anda untuk memverifikasi detail dan pengaturan akun Anda.

Jika Anda menghubungi Dukungan Apple dengan masalah Mail, Anda mungkin akan diminta untuk memilih kotak centang Aktivitas Koneksi Log di jendela. Jika dipilih, informasi diagnostik akan terkumpul saat Mail digunakan, untuk membantu Dukungan Apple memahami dan mengatasi masalah Mail, periksa masalah pada komputer Anda dan bantu tingkatkan produk dan layanan Apple. Informasi diagnostik yang terkumpul berisi informasi pribadi, termasuk konten pesan email Anda. Jika terkirim ke Apple, informasi ini akan diunakan Apple sesuai dengan kebijakan privasinya dan tidak akan dibagi dengan perusahaan lain mana pun.

Untuk melihat folder file log di Finder, klik Tampilkan Log in jendela Dokter Koneksi. Setelah file log diberikan ke Dukungan Apple, batal pilih kotak centang untuk mematikan log aktivitas koneksi. Mungkin Anda ingin menghapus file log karena berukuran besar.

Pelajari lebih lanjut mengenai Kebijakan Privasi Apple di www.apple.com/legal/privacy/id/.
