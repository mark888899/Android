Definir estilos de serie de datos

Está creando un estilo de gráfica nuevo o modificado con menos series de datos que el estilo de gráfica con el que empezó. Eso significa que hay estilos de serie de datos (combinaciones de atributos de color, relleno, trazo y sombra) que no se están utilizando. Seleccione una de las siguientes opciones para determinar cómo se gestionarán dichos estilos de serie de datos adicionales.

  • Todos los estilos de serie: Utiliza los estilos de serie definidos por el usuario y conserva los estilos anteriores para las series no definidas por el usuario. Para un nuevo estilo de gráfica, por ejemplo, hay seis series de datos, por lo que si ha definido tres, se conservarán tres de los estilos anteriores.

  • Solo los estilos de serie visibles: Utiliza únicamente los estilos de serie definidos por el usuario. Si aplica el estilo de gráfica a una gráfica con más series, los estilos de serie se repetirán.
