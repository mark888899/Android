Symboles de raccourcis clavier

Vous pouvez utiliser les raccourcis clavier (plusieurs touches sur lesquelles vous appuyez simultanément) pour accomplir rapidement de nombreuses tâches courantes, telles que la sélection de texte et la manipulation d’objets.

De nombreuses options de menu incluent leur raccourci clavier, comme ⌘N, où le symbole représente une touche de modification sur votre clavier.

Les symboles des touches de modification sont répertoriés dans le tableau ci-dessous. Pour obtenir une liste de tous les raccourcis clavier de Numbers, y compris ceux ne figurant pas dans les menus, consultez la section Raccourcis clavier associés à Pages pour Mac.
