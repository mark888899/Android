Definir estilos de séries de dados

Você está criando um estilo de gráfico novo ou modificado, com menos séries de dados do que o estilo de gráfico com o qual começou. Isso significa que há estilos de séries de dados sobressalentes (combinações de cores, preenchimento, traço e atributos de sombra) que não estão sendo usados. Escolha um dos seguintes para determinar como esses estilos de séries de dados sobressalentes devem ser administrados.

  • Todos os estilos de série: Usa os estilos de dados que você definiu e retém os estilos anteriores para as séries que você não definiu. Para um novo estilo de gráfico, por exemplo, há seis séries de dados. Portanto, se você definiu três, significa que três dos estilos anteriores serão retidos.

  • Apenas estilos de séries visíveis: Usa apenas os estilos de série que você definiu. Se você aplicar o estilo de gráfico a um gráfico com mais séries, os estilos de série se repetirão.
