Salvare un documento in un altro formato

Puoi esportare un documento di Pages per salvarne una copia in un altro formato: PDF, Microsoft Word, TXT (solo testo), ePub o Pages '09. L'esportazione di un documento è utile quando invii un documento a persone che potrebbero utilizzare software diversi dal tuo. Qualsiasi modifica che apporti al documento esportato non influenza l'originale.

Nota: Non puoi esportare i documenti di layout pagina nel formato solo testo o ePub. Per vedere se il documento è un documento di layout pagina, fai clic su File nella barra dei menu nella parte superiore dello schermo. Se vedi il comando ”Converti in elaborazione testi”, si tratta di un documento di layout pagina.

Esportare in un formato diverso

  1. Specificare le impostazioni per il formato:

    • PDF: Fai clic sul menu a comparsa “Qualità dell'immagine”, quindi scegli una qualità di immagine (più alta è la qualità dell'immagine, maggiore sarà la dimensione del file della copia esportata).

    • Word: Fai clic sul triangolo di apertura accanto a “Opzioni avanzate” e scegli un formato.

    • Solo testo: Puoi esportare solo i documenti di elaborazione testi per questo formato perché i documenti con layout pagina non hanno il corpo del testo.

    • ePub: Inserisci il nome del titolo e dell'autore. Se desideri che il libro appaia in una categoria particolare, fai clic sul menu a comparsa “Categoria principale”, quindi scegli una categoria. Per impostare la lingua e la copertina del libro e se i font devono essere incorporati, fai clic sul triangolo accanto a “Opzioni avanzate”.

    • Pages ’09: Per impostare una password, seleziona la casella di controllo “Richiedi password per aprire”, quindi inserisci le informazioni richieste.

    Nota: La password eventualmente presente nel file originale viene applicata anche alla copia. Per aprire la copia, i destinatari devono conoscere la password. Per la copia puoi tenere la password originale oppure assegnarle una password diversa. Non tutti i formati possono essere protetti da password.

  2. Fai clic su Successivo, quindi inserisci un nome per il documento.

    L'estensione del file (per esempio .pdf o .docx) viene aggiunta automaticamente al nome del documento.

  3. Inserire una o più etichette (opzionale).

  4. Fai clic sul menu a comparsa Dove, scegli dove salvare il documento, quindi fai clic su Esporta.

    Per visualizzare altre posizioni, fai clic sul pulsante freccia vicino al campo “Salva col nome”.
