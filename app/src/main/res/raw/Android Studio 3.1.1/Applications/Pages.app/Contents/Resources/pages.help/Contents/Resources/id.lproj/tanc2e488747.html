Menyimpan dokumen dalam format lain

Anda dapat mengekspor dokumen Pages untuk menyimpan salinannya dalam format lain—PDF, Microsoft Word, teks biasa (TXT), ePub, atau Pages ‘09. Mengekspor dokumen berguna saat Anda mengirimkan dokumen ke orang lain yang mungkin menggunakan perangkat lunak yang berbeda dengan Anda. Setiap perubahan yang dilakukan pada dokumen yang diekspor tidak memengaruhi versi asli.

Catatan: Anda tidak dapat mengekspor dokumen tata letak halaman ke teks biasa atau format ePub. Untuk melihat apakah dokumen Anda merupakan dokumen tata letak halaman, klik File di bar menu pada bagian atas layar. Jika Anda melihat perintah Konversi ke Pemrosesan Kata, ini adalah dokumen tata letak halaman.

Mengekspor ke format lain

  1. Tetapkan pengaturan untuk format:

    • PDF: Klik menu pop-up Kualitas Gambar, lalu pilih kualitas gambar (semakin baik kualitas gambar, semakin besar ukuran file salinan yang diekspor).

    • Word: Klik segitiga pengungkapan di samping Pilihan Lanjutan, lalu pilih format.

    • Teks Biasa: Anda hanya dapat mengekspor dokumen pemrosesan kata ke format ini karena dokumen tata letak halaman tidak memiliki teks isi.

    • ePub: Masukkan judul dan nama penulis. Jika Anda ingin agar buku Anda muncul di kategori tertentu, klik menu pop-up Kategori Utama, lalu pilih kategori. Untuk mengatur bahasa dan sampul buku serta mengatur apakah akan menyisipkan font, klik segitiga pengungkapan di samping Pilihan Lanjutan.

    • Pages ‘09: Untuk mengatur kata sandi, pilih kotak centang "Memerlukan kata sandi untuk membuka," lalu masukkan informasi yang diminta.

    Catatan: Jika file asli memiliki kata sandi, kata sandi juga diterapkan ke file salinan, dan penerima harus mengetahui kata sandi untuk membuka salinan. Anda dapat menyimpan kata sandi asli atau menetapkan kata sandi yang berbeda untuk salinan. Tidak semua format dapat dilindungi kata sandi.

  2. Klik Berikutnya, lalu ketik nama untuk dokumen.

    Ekstensi nama file (.pdf atau .docx, misalnya) ditambahkan ke nama dokumen secara otomatis.

  3. Masukkan satu atau beberapa label (opsional).

  4. Klik menu pop-up Tempat, pilih tempat untuk menyimpan dokumen, lalu klik Ekspor.

    Untuk melihat lokasi lainnya, klik tombol panah di samping bidang Simpan Sebagai.
