Envoyer une copie d’un document

Vous pouvez envoyer une copie d’un document que les destinataires peuvent ouvrir et modifier s’ils le souhaitent sur leurs propres ordinateurs et appareils. Leurs modifications n’altèrent pas le document original ni toute autre copie du document. Vous pouvez envoyer une copie au format Pages ou dans un autre format (PDF, Microsoft Word ou ePub).

Si le fichier d’origine est protégé par un mot de passe, ce dernier est également appliqué à la copie. Les destinataires doivent alors saisir le mot de passe pour ouvrir la copie. Vous pouvez conserver le mot de passe d’origine ou attribuer un autre mot de passe à la copie.

Envoyer une copie dans un autre format

  1. Cliquez sur un format, puis définissez les réglages :

    • Pages : Cochez la case et ajoutez un mot de passe si vous voulez limiter l’accès à la copie.

    • PDF : Cliquez sur le menu local Qualité de l’image, puis choisissez une qualité d’image (plus la qualité est bonne, plus la copie est volumineuse).

    • Word : Cliquez sur le triangle d’affichage en regard de la mention Options avancées, puis choisissez un format.

    • ePub : Saisissez le titre et le nom de l’auteur. Si vous voulez que votre livre apparaisse dans une catégorie spécifique, cliquez sur le menu local Catégorie principale, puis choisissez une catégorie. Pour définir la langue et la couverture, et pour indiquer si les polices sont intégrées, cliquez sur le triangle d’affichage à côté de la mention Options avancées. Vous ne pouvez pas définir de mot de passe pour les fichiers ePub.

  2. Cliquez sur Suivant, puis fournissez les informations requises (par exemple, une adresse e-mail si vous envoyez un e-mail).

    Si vous n’êtes pas connecté au compte de la méthode choisie, vous êtes invité à le faire avant de poursuivre. Pour AirDrop, le document est envoyé lorsque le destinataire l’accepte.
