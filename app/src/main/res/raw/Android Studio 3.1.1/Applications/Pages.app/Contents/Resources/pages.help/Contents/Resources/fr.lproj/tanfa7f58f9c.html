Définir les styles des séries de données

Vous créez un style de graphique avec moins de séries de données que le style de graphique duquel vous êtes parti. Cela signifie qu’il y a des styles de série de données (combinaisons d’attributs de couleur, de remplissage, de trait et d’ombre) supplémentaires qui ne sont pas utilisés. Choisissez l’une des options suivantes pour définir comment ces styles de séries de données supplémentaires sont gérés.

  • Tous les styles de série : Utilise les styles de série que vous avez définis et conserve les styles précédents pour les séries que vous n’avez pas définies. Pour un nouveau style de graphique, par exemple, il y a six séries de données ; ainsi, si vous en avez défini trois, trois des styles précédents sont conservés.

  • Uniquement les styles de série visibles : Utilise uniquement les styles de séries que vous avez définis. Si vous appliquez le style de graphique à un graphique avec plus de séries, les styles de séries se répètent.
