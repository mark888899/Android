Enregistrer un document dans un autre format

Vous pouvez exporter un document Pages pour en enregistrer une copie dans un autre format, tel que PDF, Microsoft Word, texte (.txt), ePub ou Pages ‘09. L’exportation d’un document est utile pour envoyer le document à des personnes qui n’utilisent pas le même logiciel que vous. Les modifications apportées au document exporté n’ont aucun impact sur le fichier d’origine.

Remarque : Vous ne pouvez pas exporter de documents de disposition aux formats texte et ePub. Pour savoir si votre document est un document de mise en page, cliquez sur Fichier dans la barre des menus en haut de l’écran. Si vous voyez la commande Convertir en traitement de texte, il s’agit d’un document de mise en page.

Exporter dans un autre format

  1. Pour définir les réglages liés au format :

    • PDF : Cliquez sur le menu local Qualité de l’image, puis choisissez une qualité d’image (plus la qualité est bonne, plus la copie exportée est volumineuse).

    • Word : Cliquez sur le triangle d’affichage en regard de la mention Options avancées, puis choisissez un format.

    • Format texte : Vous ne pouvez exporter que les documents de traitement de texte dans ce format, car les documents de disposition n’ont pas de corps de texte.

    • ePub : Saisissez le titre et le nom de l’auteur. Si vous voulez que votre livre apparaisse dans une catégorie spécifique, cliquez sur le menu local Catégorie principale, puis choisissez une catégorie. Pour définir la langue et la couverture, et pour indiquer si les polices sont intégrées, cliquez sur le triangle d’affichage à côté de la mention Options avancées.

    • Pages ’09 : Pour définir un mot de passe, cochez la case « Exiger un mot de passe pour l’ouverture », puis saisissez les informations demandées.

    Remarque : Si le fichier d’origine est protégé par un mot de passe, ce dernier est également appliqué à la copie. Les destinataires doivent alors saisir le mot de passe pour ouvrir la copie. Vous pouvez conserver le mot de passe d’origine ou attribuer un autre mot de passe à la copie. Certains formats ne peuvent pas être protégés par un mot de passe.

  2. Cliquez sur Suivant, puis saisissez un nom pour le document.

    L’extension de nom de fichier (.pdf ou .docx par exemple) est automatiquement ajoutée au nom du document.

  3. Saisissez un ou plusieurs tags (facultatif).

  4. Cliquez sur le menu local Enregistrer sous, indiquez l’emplacement où vous voulez enregistrer le document, puis cliquez sur Exporter.

    Pour afficher d’autres emplacements, cliquez sur la flèche en regard du champ Enregistrer sous.
