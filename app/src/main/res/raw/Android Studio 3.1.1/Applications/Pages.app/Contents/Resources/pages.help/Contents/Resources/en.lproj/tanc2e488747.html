Save a document in another format

You can export a Pages document to save a copy of it in another format—PDF, Microsoft Word, plain text (TXT), ePub, or Pages ‘09. Exporting a document is useful when you’re sending the document to people who may use different software from you. Any changes you make to the exported document don’t affect the original.

Note: You can’t export page layout documents to plain text or ePub formats. To see if your document is a page layout document, click File in the menu bar at the top of the screen. If you see the command Convert to Word Processing, it’s a page layout document.

Export to another format

  1. Specify settings for the format:

    • PDF: Click the Image Quality pop-up menu, then choose an image quality (the higher the image quality, the larger the file size of the exported copy).

    • Word: Click the disclosure triangle next to Advanced Options and choose a format.

    • Plain Text: You can export only word-processing documents to this format because page layout documents don't have body text.

    • ePub: Enter the title and author name. If you want your book to appear in a particular category, click the Primary Category pop-up menu, then choose a category. To set the language and book cover and set whether fonts are embedded, click the disclosure triangle next to Advanced Options.

    • Pages ‘09: To set a password, select the “Require password to open” checkbox, then enter the requested information.

    Note: If the original file has a password, it applies to the copy as well, and recipients must know the password to open the copy. You can keep the original password or assign a different password to the copy. Not all formats can be password protected.

  2. Click Next, then type a name for the document.

    The filename extension (.pdf or .docx, for example) is automatically appended to the document name.

  3. Enter one or more tags (optional).

  4. Click the Where pop-up menu, choose where to save the document, then click Export.

    To see more locations, click the arrow button next to the Save As field.
