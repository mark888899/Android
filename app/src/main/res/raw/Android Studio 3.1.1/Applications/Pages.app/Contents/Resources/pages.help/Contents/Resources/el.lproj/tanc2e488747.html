Αποθήκευση εγγράφου σε διαφορετική μορφή

Μπορείτε να εξάγετε ένα έγγραφο Pages για να αποθηκεύσετε ένα αντίγραφό του σε άλλη μορφή—PDF, Microsoft Word, απλό κείμενο (TXT), ePub, ή Pages ‘09. Η εξαγωγή εγγράφου είναι χρήσιμη όταν στέλνετε το έγγραφο σε άλλα άτομα που ενδέχεται να χρησιμοποιούν διαφορετικό λογισμικό από εσάς. Τυχόν αλλαγές που πραγματοποιείτε στο έγγραφο εξαγωγής δεν επηρεάζουν το πρωτότυπο.

Σημείωση: Δεν είναι δυνατή η εξαγωγή εγγράφων διάταξης σελίδων σε μορφή απλού κειμένου ή ePub. Για να δείτε αν το έγγραφό σας είναι έγγραφο διάταξης σελίδων, κάντε κλικ στο «Αρχείο» στη γραμμή μενού στο πάνω μέρος της οθόνης. Αν εμφανιστεί η εντολή «Μετατροπή σε επεξεργασία κειμένου», τότε το έγγραφο είναι έγγραφο διάταξης σελίδων.

Εξαγωγή σε άλλη μορφή

  1. Καθορισμός ρυθμίσεων για τη μορφή:

    • PDF: Κάντε κλικ στο αναδυόμενο μενού «Ποιότητα εικόνας» και μετά επιλέξτε μια ποιότητα εικόνας (όσο υψηλότερη η ποιότητα, τόσο μεγαλύτερο θα είναι το μέγεθος του εξαγόμενου αντιγράφου).

    • Λέξη: Κάντε κλικ στο τρίγωνο αποκάλυψης δίπλα στις «Προηγμένες επιλογές» και επιλέξτε μια μορφή.

    • Απλό κείμενο: Μπορείτε να εξάγετε μόνο έγγραφα επεξεργασίας κειμένου σε αυτήν τη μορφή διότι τα έγγραφα διάταξης σελίδων δεν έχουν κύριο τμήμα κειμένου.

    • ePub: Εισαγάγετε τον τίτλο και το όνομα του συγγραφέα. Αν θέλετε το βιβλίο σας να εμφανίζεται σε μια συγκεκριμένη κατηγορία, κάντε κλικ στο αναδυόμενο μενού «Κύρια κατηγορία» και μετά επιλέξτε μια κατηγορία. Για να καθορίσετε τη γλώσσα και το εξώφυλλο βιβλίου και να ρυθμίσετε αν οι γραμματοσειρές θα είναι ενσωματωμένες, κάντε κλικ στο τρίγωνο αποκάλυψης δίπλα στις «Προηγμένες επιλογές».

    • Pages ’09: Για να καθορίσετε ένα συνθηματικό, επιλέξτε το πλαίσιο επιλογής «Απαίτηση συνθηματικού για άνοιγμα» και μετά εισαγάγετε τις απαιτούμενες πληροφορίες.

    Σημείωση: Αν το αρχικό αρχείο προστατεύεται με συνθηματικό, αυτό ισχύει και για το αντίγραφο, και οι παραλήπτες πρέπει να γνωρίζουν το συνθηματικό για να ανοίξουν το αντίγραφο. Μπορείτε να διατηρήσετε το αρχικό συνθηματικό ή να εκχωρήσετε διαφορετικό συνθηματικό στο αντίγραφο. Κάποιες μορφές αρχείων δεν επιτρέπουν την προστασία με συνθηματικό.

  2. Κάντε κλικ στο «Επόμενο» και μετά πληκτρολογήστε ένα όνομα για το έγγραφο.

    Η επέκταση ονόματος αρχείου (για παράδειγμα, .pdf ή .docx) προστίθεται αυτόματα στο όνομα του εγγράφου.

  3. Εισαγάγετε μία ή περισσότερες ετικέτες (προαιρετικό).

  4. Κάντε κλικ στο αναδυόμενο μενού «Τοποθεσία», επιλέξτε πού θέλετε να αποθηκευτεί το έγγραφο και μετά κάντε κλικ στην «Εξαγωγή».

    Για να εμφανιστούν περισσότερες τοποθεσίες, κάντε κλικ στο κουμπί βέλους δίπλα στο πεδίο «Αποθήκευση ως».
