Senden einer Kopie eines Dokuments

Sie können eine Kopie eines Dokuments senden, die von den Empfängern auf deren eigenen Computern und Geräten geöffnet und bearbeitet werden kann. Ihre Änderungen haben keine Auswirkungen auf das Originaldokument oder andere Kopien des Dokuments. Sie können die Kopie als Pages-Dokument oder in einem anderen Format – PDF, Microsoft Word oder ePub – senden.

Wenn die Originaldatei mit einem Passwort geschützt ist, wird dieses auch auf die Kopie angewendet. Die Empfänger müssen das Passwort also kennen, um die Kopie öffnen zu können. Sie können für die Kopie das Originalpasswort beibehalten oder ein anderes Passwort zuweisen.

Senden eines Dokuments in einem anderen Format

  1. Klicken Sie auf ein Format und geben Sie Folgendes an:

    • Pages: Markieren Sie das Feld und fügen Sie ein Passwort hinzu, wenn Sie den Zugriff auf die Kopie beschränken möchten.

    • PDF: Öffnen Sie das Einblendmenü „Bildqualität“ und wählen Sie eine Bildqualität aus. (Je höher die Bildqualität ist, desto größer wird die exportierte Kopie.)

    • Word: Klicken Sie auf das Dreiecksymbol neben „Erweiterte Optionen“ und wählen Sie ein Format.

    • ePub: Geben Sie den Titel und den Namen des Autors ein. Soll Ihr Buch in einer bestimmten Kategorie erscheinen, öffnen Sie das Einblendmenü „Primäre Kategorie“ und wählen Sie eine Kategorie aus. Klicken Sie auf das Dreiecksymbol neben „Erweiterte Optionen“, um die Sprache und den Bucheinband festzulegen und um anzugeben, ob die Schriften eingebettet werden sollen. Für eine ePub-Datei kann kein Passwort festgelegt werden.

  2. Klicken Sie auf „Weiter“ und geben Sie die benötigten Informationen ein (für den Versand per E-Mail zum Beispiel die E-Mail-Adresse).

    Wenn Sie bei dem Account für die gewählte Methode nicht angemeldet sind, werden Sie aufgefordert, sich anzumelden. Bei AirDrop wird das Dokument gesendet, wenn es der Empfänger akzeptiert.
