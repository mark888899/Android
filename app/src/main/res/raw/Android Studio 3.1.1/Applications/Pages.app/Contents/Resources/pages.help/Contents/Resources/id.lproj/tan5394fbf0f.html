Mengirim salinan dokumen

Anda dapat mengirim salinan dokumen yang dapat dibuka dan dimodifikasi penerima seperti yang diinginkan pada komputer dan perangkat mereka. Perubahan mereka tidak akan memengaruhi dokumen asli atau salinan lain dokumen. Anda dapat mengirim salinan sebagai dokumen Pages atau dalam format lainnyaâ€”PDF, Microsoft Word, atau ePub.

Jika file asli memiliki kata sandi, kata sandi juga diterapkan ke file salinan, dan penerima harus mengetahui kata sandi untuk membuka salinan. Anda dapat menyimpan kata sandi asli atau menetapkan kata sandi yang berbeda untuk salinan.

Mengirim salinan dalam format lain

  1. Klik format, lalu tentukan pengaturan:

    • Pages: Pilih kotak centang dan tambahkan kata sandi jika Anda ingin membatasi akses ke salinan.

    • PDF: Klik menu pop-up Kualitas Gambar, lalu pilih kualitas gambar (semakin tinggi kualitas gambar, maka semakin besar ukuran file salinan).

    • Word: Klik segitiga pengungkapan di samping Pilihan Lanjutan, lalu pilih format.

    • ePub: Masukkan judul dan nama penulis. Jika Anda ingin agar buku Anda muncul di kategori tertentu, klik menu pop-up Kategori Utama, lalu pilih kategori. Untuk mengatur bahasa dan sampul buku serta mengatur apakah akan menyisipkan font, klik segitiga pengungkapan di samping Pilihan Lanjutan. Anda tidak dapat mengatur kata sandi untuk file ePub.

  2. Klik Berikutnya, lalu berikan informasi yang diminta (alamat email jika Anda mengirimkan email, misalnya).

    Jika Anda belum masuk ke akun untuk metode yang dipilih, Anda akan diminta untuk masuk sebelum melanjutkan. Untuk AirDrop, dokumen dikirim saat penerima menerimanya.
