데이터 시리즈 스타일 정의하기

사용자가 지정한 차트 스타일보다 적은 데이터 시리즈로 새로운 차트 스타일 또는 수정한 차트 스타일을 생성할 수 있습니다. 다시 말해 사용하는 데이터 시리즈 스타일 외에 추가적인 데이터 시리즈 스타일(색상, 채우기, 선 및 그림자 속성의 조합)을 사용할 수 있습니다. 다음 중 하나를 수행하여 추가 데이터 시리즈 스타일을 처리하는 방법을 정의하십시오.

  • 모든 시리즈 스타일: 사용자가 정의한 시리즈 스타일을 사용하며 사용자가 정의하지 않은 이전 스타일은 유지됩니다. 예를 들어 새로운 차트 스타일에 6개의 데이터 시리즈가 있는 경우 사용자가 3개의 스타일을 정의하면 이전 3개의 스타일은 유지됩니다.

  • 볼 수 있는 시리즈 스타일만: 사용자가 정의한 시리즈 스타일만 사용합니다. 더 많은 시리즈가 있는 차트에 해당 차트 스타일을 적용하면 시리즈 스타일이 반복됩니다.
