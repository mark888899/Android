Sichern eines Dokuments in einem anderen Format

Sie können ein Pages-Dokument exportieren, um eine Kopie in einem anderen Format zu sichern. Verfügbar sind die Optionen PDF, Microsoft Wort, reiner Text (TXT), ePub oder Pages ‘09. Der Export eines Dokuments empfiehlt sich, wenn Sie es an Personen senden, die möglicherweise mit anderer Software als Sie arbeiten. Änderungen, die am exportierten Dokument vorgenommen werden, haben keine Auswirkungen auf das Original.

Hinweis: Sie können Seitenlayoutdokumente nicht als reinen Text oder im ePub-Format exportieren. Um festzustellen, ob Sie mit einem Seitenlayoutdokument arbeiten, öffnen Sie das Menü „Ablage“, das sich oben am Bildschirmrand in der Menüleiste befindet. Wird der Befehl „In Textverarbeitung konvertieren“ angezeigt, handelt es sich um ein Seitenlayoutdokument.

In ein anderes Format exportieren

  1. Formateinstellungen festlegen

    • PDF: Öffnen Sie das Einblendmenü „Bildqualität“ und wählen Sie eine Bildqualität aus. (Je höher die Bildqualität ist, desto größer wird die exportierte Datei.)

    • Word: Klicken Sie auf das Dreiecksymbol neben „Erweiterte Optionen“ und wählen Sie ein Format.

    • Reiner Text: Sie können nur Textverarbeitungsdokumente in dieses Format exportieren, weil Seitenlayoutdokumente keine Fließtext enthalten.

    • ePub: Geben Sie den Titel und den Namen des Autors ein. Soll Ihr Buch in einer bestimmten Kategorie erscheinen, öffnen Sie das Einblendmenü „Primäre Kategorie“ und wählen Sie eine Kategorie aus. Klicken Sie auf das Dreiecksymbol neben „Erweiterte Optionen“, um die Sprache und den Bucheinband festzulegen und um anzugeben, ob die Schriften eingebettet werden sollen.

    • Pages ‘09: Markieren Sie das Feld „Zum Öffnen Passwort anfordern“ und geben Sie die erforderlichen Informationen ein.

    Hinweis: Wenn die Originaldatei mit einem Passwort geschützt ist, wird dieses auch auf die Kopie angewendet. Die Empfänger müssen das Passwort also kennen, um die Kopie öffnen zu können. Sie können für die Kopie das Originalpasswort beibehalten oder ein anderes Passwort zuweisen. Nicht alle Formate können mit einem Passwort geschützt werden.

  2. Klicken Sie auf „Weiter“ und geben Sie einen Namen für das Dokument ein.

    Das Dateisuffix (z. B. .pdf oder .epub) wird automatisch an den Dateinamen angehängt.

  3. Fügen Sie ein oder mehrere Tags hinzu (optional).

  4. Klicken Sie auf das Einblendmenü „Ort“ und wählen Sie den gewünschten Speicherort für das Dokument. Klicken Sie anschließend auf „Exportieren“.

    Klicken Sie auf die Pfeiltaste neben dem Feld „Sichern unter“, um weitere Speicherorte anzuzeigen.
